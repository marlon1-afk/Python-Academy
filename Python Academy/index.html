Wie ein Programmierer denken: Programmieren ist wie ein Kochrezept schreiben. Der Computer macht stur genau das, was du Zeile für Zeile aufschreibst. Wenn etwas nicht klappt, nennt man den Fehler einen "Bug".

Schubladen für Daten

Eine Variable ist eine Schublade mit Namen. Du legst ein Wort oder eine Zahl hinein, um es für später aufzuheben.

Python
name = "Anna" # Wort in Schublade "name" packen alter = 21 # Zahl in Schublade "alter" packen print(name) # Guck in die Schublade und zeig es an
Bildschirm-Ausgabe
Anna

Sachen ausdrucken

Der Befehl print() ist wie ein kleiner Drucker. Alles in den Klammern wird auf den Bildschirm geschrieben.

Python
print("Hallo Welt!")
Bildschirm-Ausgabe
Hallo Welt!

Den Nutzer fragen

Mit input() stellt der Computer dem Nutzer eine Frage und wartet, bis dieser etwas eintippt und Enter drückt.

Python
farbe = input("Deine Lieblingsfarbe? ") print("Cool,", farbe)
Bildschirm-Ausgabe
Deine Lieblingsfarbe? Blau Cool, Blau

Wege entscheiden

Mit if (Wenn) und else (Ansonsten) biegt der Computer ab, je nachdem was stimmt. Wie ein Türsteher im Club.

Python
alter = 15 if alter >= 18: print("Rein in den Club!") else: print("Du musst leider draußen warten.")
Bildschirm-Ausgabe
Du musst leider draußen warten.

Listen [ ]

Was ist, wenn du nicht nur eine Sache, sondern ganz viele Sachen in eine Variable packen willst? Dafür gibt es Listen! Stell es dir wie einen Einkaufszettel vor.

Wichtig: Der Computer fängt immer bei 0 an zu zählen! Die erste Sache auf der Liste ist also die Nummer 0.
Python
einkauf = ["Apfel", "Banane", "Milch"] print(einkauf[0]) # Zeige die Sache an Stelle 0 (die erste Sache) print(einkauf[2]) # Zeige die Sache an Stelle 2 (die dritte Sache)
Bildschirm-Ausgabe
Apfel Milch

while-Schleifen (Wiederholungen)

Mit einer Schleife kannst du den Computer zwingen, eine Aufgabe immer und immer wieder zu wiederholen, so lange, bis eine Bedingung erfüllt ist.

Python
zaehler = 1 while zaehler < = 3: # Solange der Zähler kleiner oder gleich 3 ist: print("Wiederholung Nummer", zaehler) zaehler = zaehler + 1
Bildschirm-Ausgabe
Wiederholung Nummer 1 Wiederholung Nummer 2 Wiederholung Nummer 3

Wie man eine KI richtig benutzt

Eine KI kann dir genial beim Programmieren lernen helfen. Stell dir die KI wie einen extrem fleißigen, aber etwas unschlauen Praktikanten vor.

Die 3 goldenen KI-Regeln

1. Häppchen-Taktik
Frag die KI niemals nach einem riesigen, fertigen Programm. Frag immer nach kleinen Teilen.
2. Sag der KI, wer sie ist
Gib der KI eine Rolle, z.B. „Du bist ein geduldiger Lehrer für Anfänger“.
3. Zeig ihr deinen Code
Kopiere fehlerhaften Code komplett hinein und nenne die exakte Fehlermeldung.
